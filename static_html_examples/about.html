Purpose: This website has been designed for the F21NA course at HWU. Its goal is to offer a chat service for discussion on general topics.


Links:

  • Index page. This page allows a user to authenticate itself to the chat service
  • 'Main' page of the application. Uses a Single-Page design.
  • base.css base css stylesheet
  • chat.css chat css stylesheet
  • index.js Javascript functions for index page
  • animation.js Javascript functions to animate panels on chat page

On the chat page, one can send a message to a specific channel using the text area at the bottom.

Also, one may creates channels with the text area on the left panel.

Channels can be joined, left and deleted using specific buttons (joined by clicking on the channel, left by clicking the green window and deleted by clicking the red trash)

Red badges on the right side indicates the number of unread messages in that channel.

connected users are displayed on the right panel

Finally, one can disconnect for the chat using the 'open-window' button on the top-left panel.


Because this application contains only two pages, everybody was working as a web designer on each page.



External libraries: jQuery and Font Awesome

Using jQuery for such simple pages can be surprising but it will be largely used when we will add interactions with the server (See Stage 2)

Also, CSS Stylesheets from Font Awesome raise a lots of error on CSS Validator. These errors are beyond our will!



Our design takes advantage of CSS3 transitions and HTML5 attributes such as 'autofocus', 'spellcheck' or 'autocomplete'
